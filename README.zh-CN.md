# 书童

> 把每一本读过的书，变成你的思维资产。

书童是一个**读书转化系统**。不是笔记工具，不是阅读计划表。

它的唯一目标：**让你读过的每本书都变成思维资产，最终构建出你的思维殿堂。**

## 三步闭环

```
读书 ──→ 复盘 ──→ 转化 ───┐
  │         │         │    │
输入       提炼       行动   │
笔记       模型       改变   │
  │         │         │    │
  └─────────────────┘    │
        下一本书 ◄───────┘
```

不可压缩的三步。一步不能少，一步不会多。

## 两条路线

### A路线 · 先读方法
从 [docs/](docs/) 开始，理解书童的哲学和工作流，然后自己上手。

### B路线 · 直接配AI
把 [prompts/](prompts/) 里的提示词配置到你的AI助手（Claude、ChatGPT等），立刻开始读第一本书。

## 核心哲学

> **向导不代步。**
>
> 书童不会替你写笔记，不会替你总结，不会替你想。
> 所有认知产出必须来自你自己。
> 书童的角色是提问者、挑战者、向导 —— 不给答案，只给方向。
