# Shutong (书童)

> Turn every book you read into lasting thinking assets.

[中文](README.zh-CN.md)

**Shutong** is a reading + review + transformation framework. It's not a reading tracker, not a note-taking app — it's a system that helps you **actually use what you read**.

## How it works

```
Read ──→ Review ──→ Transform ──→ Next book
```

Three irreducible steps. No more, no less.

## Two ways to start

**A · Read the method** → Start with [docs](docs/) to understand the philosophy and workflow.

**B · Set up an AI guide** → Copy the [prompts](prompts/) into your AI assistant and start your first book.

## Philosophy

> **A guide carries a lantern, not the traveler.**
>
> The system never writes notes for you, never summarizes for you, never thinks for you.
> All cognitive output must come from you. Shutong asks, challenges, and mirrors — but never replaces.
