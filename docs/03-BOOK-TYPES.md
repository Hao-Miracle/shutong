# 4种书型策略

不同书需要不同读法。以下是书童对非虚构类书籍的分类策略。

---

## 1. 工具书/方法论

如《原则》《影响力》《非暴力沟通》

**读法：** 精读 + 反复咀嚼关键章节
**速度：** 慢，一章可能读多次
**笔记重点：** 框架、清单、决策模型
**互动风格：** 每章结束后用问题复盘讨论

## 2. 叙事书/传记

如《史蒂夫·乔布斯传》《刻意练习》

**读法：** 通读为主 + 关键事件深挖
**速度：** 中快，保持流畅性
**笔记重点：** 关键时刻、决策逻辑、人物模式
**互动风格：** 按时间线推进，穿插"如果是你怎么做"

## 3. 理论书/学术

如《思考快与慢》《创新者的窘境》

**读法：** 概念先行 → 实例验证 → 批判思考
**速度：** 中慢，概念处必须停
**笔记重点：** 定义、实验证据、边界条件
**互动风格：** 每个核心概念用费曼式追问

## 4. 散文集/随笔

如《纳瓦尔宝典》

**读法：** 跳跃式，独立篇章可单独消化
**速度：** 灵活，好的一篇反复读
**笔记重点：** 金句、反直觉洞察、个人反思
**互动风格：** 挑最有共鸣的篇深入

---

## 如何判断书型

拿到一本书后，看三点：

1. **作者的写作目的**：是教你怎么做（工具书），还是讲一个故事（叙事），还是论证一个理论（学术），还是分享思考（散文）？
2. **章节结构**：章节之间有递进关系吗？还是各自独立？
3. **案例比重**：书中案例是用来辅助说明概念的，还是故事本身就是核心？

确定书型后，AI会根据书型和具体内容定制阅读方案。
