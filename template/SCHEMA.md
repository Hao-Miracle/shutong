# SCHEMA.md · 结构规范

> 思维殿堂知识库的目录结构和文件命名规范。

```
知识库/
├── raw/                   ← 原始PDF（只存不碰）
├── 《书名》/              ← 每本书一个文件夹
│   ├── 笔记.md           ← 你写：读书笔记
│   ├── 反思.md           ← 你写：复盘反思
│   ├── 交互记录.md       ← AI自动维护：对话记录
│   └── 行动.md           ← 你写：行动计划
├── 概念图谱/              ← 跨书概念网络
│   ├── 概念/             ← 每个概念一个文件
│   └── 格栅/             ← 概念间关系
├── 实践工坊/              ← 行动验证
│   └── 项目/             ← 每个行动项目一个文件
├── 00-宫殿大门/           ← 入口 + 成就
├── MIND_PALACE.md         ← 殿堂总览
├── SCHEMA.md              ← 本文件
└── log.md                 ← 读书日志
```
