# 思维殿堂 · 总览

欢迎来到你的思维殿堂。

每一本你认真读完并复盘过的书，都是这座殿堂里的一块砖。
每一块砖累积起来，就形成了你的知识结构——不只是"读过什么书"，而是"我拥有什么思维工具"。

## 结构

```
raw/                  ← 原始书籍PDF（丢进去）
《书名》/            ← 每本书一个文件夹
├── 笔记.md          ← 读书笔记（你写）
├── 反思.md          ← 复盘反思（你写）
├── 交互记录.md      ← AI引导对话（自动记录）
└── 行动.md          ← 行动计划（你写）
概念图谱/             ← 跨书概念链接
实践工坊/             ← 行动验证
00-宫殿大门/          ← 系统入口 + 成就
```

## 核心流程

```
读书 → 复盘 → 转化 → 下一本
```
